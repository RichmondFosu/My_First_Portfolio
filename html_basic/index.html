Project: Basic Structure

This page is a continuation of the base_index.html project. It demonstrates how to use headings, paragraphs, and images correctly.

I’m practicing semantic HTML to make my pages accessible and easy to maintain. Proper structure helps both browsers and assistive technologies understand content.

Using clean markup also ensures that validation with tools like the W3C validator passes without errors, which is an important habit for professional front-end developers.

ALX encourages hands-on projects, so I’m typing this code myself and learning how each element works instead of just copying and pasting.
